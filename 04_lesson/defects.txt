Баг №3
Дата: 20.06.2026
Метод: contains
Описание: Метод ищет подстроку, а не один символ
Название метода contains и параметр symbol подразумевают поиск одного символа.
Однако string.index(symbol) ищет любую подстроку.
Пример: contains("SkyPro", "Pro") → True (нашёл подстроку из 3 символов)
Ожидаемое поведение: поиск только одного символа, либо переименовать параметр
Серьёзность: Средняя (вводит в заблуждение)
